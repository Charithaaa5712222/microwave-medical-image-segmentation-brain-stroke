# Microwave Medical Image Segmentation for Brain Stroke Diagnosis

This project is a Django-based medical image processing application.

Features:

- User Registration and Login
- OTSU Segmentation
- DBIM Segmentation
- Brain Stroke/Tumor Region Detection
- MySQL Database Integration (XAMPP)

Technologies:

- Python 3.8
- Django 2.1.7
- OpenCV
- NumPy
- Matplotlib
- PyMySQL
- XAMPP MySQL

Run:

1. Start Apache and MySQL in XAMPP
2. Run:

py -3.8 manage.py runserver

3. Open:

http://127.0.0.1:8000/index.html
